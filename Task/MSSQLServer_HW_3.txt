Требования к оформлению.
Работу необходимо оформить в виде одного файла с расширением sql. Имя файла должно состоять из названия группы, фамилии студента и суффикса «ЛР1».
Пример: «J31017.Иванов.ЛР2.sql»
Задание.
1.	Написать скрипт создания базы данных проекта. Имя базы должно содержать в себе фамилию студента.
2.	Написать скрипт создания таблиц. Скрипт должен содержать индексы.
3.	Написать создания внешних ключей для таблиц.
4.	Написать запросы вставки тестовых данных в таблицы (минимум по 3 записи).
Скрипт должны находится в одном файле. При запуске скрипта, база данных, таблицы и внешние ключи и запросы вставки данных должны генерироваться без ошибок.
